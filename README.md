# Cloudflare Turnstile Test

Simple page for testing Cloudflare Turnstile captcha.

## Setup

1. Get your keys from [Cloudflare Dashboard](https://dash.cloudflare.com/) → Security → Turnstile → Add a site
2. Replace `REPLACE_WITH_YOUR_SITEKEY` in `index.html` with your actual sitekey
